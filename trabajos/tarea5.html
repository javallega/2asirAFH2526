Conexión SSH

Comentarios

  • Se ha accedido a una máquina Ubuntu server desde desde un windows 10 home físico.
  • Se ha acedido a una máquina desde Ubuntu server desde un a maquina ubuntu.
  • Permisos de administrador

Pasos de conexión SSH

1

instalación de SSH en linux

Re alizamos la instalaicón de SSH en la máquina Ubuntu server con la instrucción sudo apt update y sudo apt install -y openssh-server.

2

Creacción de un usuario en linux

Creamos un usuario josevallega en linex server para acceder con ese usuario. adduser josevallega. Creamos la contraseña y damos la información que necesitemos guardar, nombre completo, etc. Con el comando sudo usermod -aG sudo nombre_usuario le damos permiso de sudo al nuevo usuario.

3

Conexión co otro usuario linus

Desde la linea de comandos de la maquina cliente Ubuntu escribimos ssh josevallega@192.168.0.105 (es la direción que tiene el servidor dentro de mi red local)

4

Conexión SSH desde un windows con putty

Una vez instalado en windows el programa putty, lo abrimos y ponemos la dirección a la que queremos conectarnos en mi caso la de ubuntu serve 192.168.0.105

Conexión de escritorio Remonot en Windows con RDP

Método sención de conecto y aceder a un escritorio en Windows.

Necesitamos

  • Habilitar escritorio remoto en W11
  • instalar Remmina en ubunto para poder conercta con el escritorio Remoto de Windows.
  • Red que se vean y compatibilidad en el protocolo TLS

Escritorio remoto Windows RDP

1

Activar co habilitar Escritorio remoto en Windows

Dentro de windows en configuración/Sistema/ escritorio Remoto/Habilitar. Habilitamos nuestro escritorio remoto.

2

Añadimos el usuario javallega

Añadimos el usuario javallega a Windows para conectarnos desde otros equipos.

3

Conectamos desde Windows

Desde el Windows cliente en mi caso un W10 Home físico vamos hacer una conexion a windows 11 físico a la direcicón 192.168.0.118

4

Accedemos al escritorio Windows.

Nos piede que si queremos entrar en la sesión de javallega ya que tenemos otra sesión abierta en w11 y si le decimos que si entramnos al escritorio remoto

5

Instalación remmina en Máquina virtual de Ubuntu

Instalamos Remmina en la máquina virtual de Ubuntú.

6

Conectamos con Remmana

Nos conectamos con Remmina a la ip de nuestro windows en mi caso 192.168.0.118 y ponemos usuario y contraseña.

7

Accedemos al escritorio Remoto

En este caso accedemos al escritorio de windows y nos pregunta que tenemos una sesión abierta con otro usuario y si queremos cerrar y abrir con nuestro usuario.
