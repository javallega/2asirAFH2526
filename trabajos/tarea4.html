Instalación de Apache en Windows

Apache HTTP Server es uno de los servidores web más populares del mundo. En esta guía aprenderás cómo instalarlo y configurarlo en sistemas Windows.

Requisitos previos

  • Windows 7 o superior (preferiblemente Windows 10/11)
  • Microsoft Visual C++ 2017 Redistributable (VC15)
  • Permisos de administrador

Pasos de instalación

1

Descargar Apache

Visita Apache Lounge (https://www.apachelounge.com) y descarga la versión más reciente para Windows (64-bit o 32-bit según tu sistema).

2

Extraer archivos

Extrae el archivo ZIP descargado en la raíz de tu disco, por ejemplo C:\. Esto creará una carpeta C:\Apache24

3

Configurar httpd.conf

Abre C:\Apache24\conf\httpd.conf con un editor de texto. Verifica ServerRoot, Listen 80, ServerName localhost:80, y DocumentRoot.

4

Instalar como servicio

Abre CMD como administrador, navega a C:\Apache24\bin\ y ejecuta: httpd.exe -k install

5

Iniciar Apache

Ejecuta httpd.exe -k start o inicia el servicio Apache2.4 desde services.msc

6

Configurar Firewall

Permite el tráfico en el puerto 80 en el Firewall de Windows

7

Verificar instalación

Abre un navegador y visita http://localhost. Deberías ver la página de prueba de Apache.

Instalación de Apache en Linux (Ubuntu)

Apache es especialmente popular en sistemas Linux. Esta guía te mostrará cómo instalarlo en Ubuntu/Debian de manera sencilla.

Requisitos previos

  • Ubuntu 20.04 o superior (o distribución basada en Debian)
  • Privilegios sudo
  • Conexión a Internet

Pasos de instalación

1

Actualizar paquetes

Abre la terminal y ejecuta: sudo apt update && sudo apt upgrade -y

2

Instalar Apache

Ejecuta el comando: sudo apt install apache2 -y

3

Verificar estado

Comprueba que Apache esté ejecutándose: sudo systemctl status apache2

4

Habilitar en el arranque

Asegura que Apache inicie automáticamente: sudo systemctl enable apache2

5

Configurar firewall

Si usas UFW, permite Apache: sudo ufw allow 'Apache' o sudo ufw allow 80/tcp

6

Verificar instalación

Abre un navegador y visita http://localhost o http://tu_ip_servidor

7

Comandos útiles

Gestiona Apache con: sudo systemctl start/stop/restart/reload apache2
