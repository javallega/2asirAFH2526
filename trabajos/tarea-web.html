Portfolio: Instalación de Apache

Guía sencilla para instalar el servidor web Apache en Windows y Ubuntu

Apache en Windows

  1. Visita Apache Lounge y descarga la última versión de Apache para Windows.
  2. Descomprime el archivo ZIP en C:\Apache24 (o la ruta predeterminada indicada en la guía).
  3. Abre la línea de comandos como Administrador.
  4. Ubica la carpeta bin de Apache:
    cd C:\Apache24\bin
  5. Para instalar Apache como servicio, ejecuta:
    httpd.exe -k install
  6. Inicia el servicio con:
    httpd.exe -k start
  7. ¡Listo! Comprueba en el navegador:
    http://localhost

Apache en Ubuntu

  1. Abre una terminal en Ubuntu.
  2. Actualiza los repositorios:
    sudo apt update
  3. Instala el servidor Apache:
    sudo apt install apache2
  4. Inicia el servicio:
    sudo systemctl start apache2
  5. Habilita Apache para que inicie con el sistema:
    sudo systemctl enable apache2
  6. Comprueba en el navegador:
    http://localhost
